Integral Tak Tentu Fungsi Aljabar dan Sifat-Sifatnya

Pada bagian pertama kita telah mengenal definisi integral dan kaitannya dengan konsep turunan. Nah kali ini kita akan belajar mengenai sifat-sifat yang dimiliki integral tak tentu sehingga kita dapat memahami karakteristik dari integral dalam soal-soal yang akan kita kerjakan.
Bentuk umum integral tak tentu adalah sebagai berikut.
dengan: $c$ adalah konstanta pengintegralan
$f(x)$ adalah integran
$F(x)$adalah fungsi integral umum dan $F(x)$ bersifat $F’(x) = f(x)$
Bagaimanakah aturan-aturan atau sifat-sifat integral tak tentu? Berikut penjelasan mengenai aturan-aturan atau sifat-sifat integral tak tentu fungsi aljabar.
  1. Rumus
    1. $\int dx=x+c$
    2. $\int$ $k$ $dx=kx+c $
    3. $\int$ $x^n$ $dx$= $\frac{1}{n+1}x^{n+1}+c$
      Dengan n bilangan rasional dan n ≠ -1
    4. $\int$ $kx^{n}dx$ =$\frac{k}{n+1}x^{n+1}+c$
      Dengan n bilangan rasional dan n ≠ -1
    5. $\int$ $(kx+c)^{n}dx$ = $\frac{1}{k(n+1)}$ $(kx+c)^{n+1}+c$
      Dengan n bilangan rasional dan n ≠ -1
  2. sifat-sifat
    1. $\int k.f(x)dx$ = $k\int f(x)dx$
    2. $\int$ {$f(x)+ g(x)$}$dx$= $\int f(x)dx$ + $\int g(x)dx $
    3. $\int$ {$f(x)- g(x)$}$dx$= $\int f(x)dx$ - $\int g(x)dx $
Untuk memperdalam pemahaman kalian mengenai sifat-sifat dari integral tak tentu coba perhatikan contoh soal dibawah ini!

Tentukan hasil integral dari: $\int{3x^2}dx$
Penyelesaian:
$\int{3x^2}dx$ = $\frac{3}{2+1}x^{2+1} + c$
     = $\frac{3}{3}x^{3} + c$
     = $x^3 +c$
Tentukan hasil integral dari: $\frac{1}{x^2} dx$
Penyelesaian:
$\frac{1}{x^2} dx$ = $\int{x^{-2}} dx$
   = $\frac{1}{-2+1}x^{-2+1} + c$
   = $\frac{1}{-1}x^{-1} + c$
   = $-x^{-1} + c$
   = $-\frac{1}{x} + c$
Tentukan hasil integral dari: $\sqrt{x^3}dx$
Penyelesaian:
$\sqrt{x^3}dx$ = $\int x^{\frac{3}{2}} dx$
    = $\frac{1}{\frac{3}{2}+1}x^{\frac{3}{2}+1}+c$
    = $\frac{1}{\frac{5}{2}}x^{\frac{5}{2}}+c $
    = $\frac{2}{5}x^{\frac{5}{2}}+c$
Tentukan hasil integral dari: $\frac{1}{\sqrt{x^3}}dx$
Penyelesaian:
$\frac{1}{\sqrt{x^3}}dx$ = $\int x^{-\frac{3}{2}} $
    = $\frac{1}{-\frac{3}{2}+1}x^{-\frac{3}{2}+1}+c$
    = $\frac{1}{-\frac{1}{2}}x^{-\frac{1}{2}}+c$
    = $\frac{-2}{\sqrt{x}}+c$

Tentukanlah nilai dari $\int(3x^6 – 2x^2 + 1) dx$
Penyelesaian:
$\int(3x^6 – 2x^2 + 1) dx$ = $\int 3x^6 dx - \int 2x^2 dx + \int 1 dx$
         = $3\int x^6 dx - 2\int x^2 dx + \int 1 dx$
         = $3[\frac{1}{6+1}x^{6+1}]-2[\frac{1}{2+1}x^{2+1}]+ x + c$
         = $3[\frac{1}{7}x^7] - 2[\frac{1}{3}x^3] + x + c$
         = $\frac{1}{7}x^7 - \frac{2}{3}x^3 + x + c$
Tentukanlah nilai dari $\int(x + 2)^2 dx$
Penyelesaian:
$\int(x + 2)^2 dx$ = $\int(x+2)(x+2)dx$
      = $\int x^2 + 4x + 4 dx$
      = $\int x^2 dx + \int 4x dx + \int 4 dx$
      = $\int x^2 dx + 4 \int x dx + \int 4 dx $
      = $\frac{1}{2+1}x^{2+1} + 4[\frac{1}{1+1}x^{1+1}] + 4x + c $
      = $\frac{1}{3}x^3 + 4[\frac{1}{2}x^2] + 4x + c$
      = $\frac{1}{3}x^3 + \frac{4}{2}x^2 + 4x + c$
      =$\frac{1}{3}x^3 + 2x^2 + 4x + c$
Carilah nilai $f(x)$ jika $f'(x)$ = $ x^3 - 4x^2 + 3$ dan $f(0) = 1$
Penyelesaian:
$f'(x) = x^3 - 4x^2 + 3$ maka $f(x) = \int x^3 - 4x^2 + 3 dx$
$f(x) = \int x^3 dx - \int 4x^2 dx + \int 3 dx$
   $= \int x^3 dx - 4\int x^2 dx + \int 3 dx $
   $= \frac{1}{3+1}x^{3+1} - 4[\frac{1}{2+1}x^{2+1}] + 3x + c$
   $= \frac{1}{4}x^4 - \frac{4}{3}x^3 + 3x + c$
→ $f(x) = \frac{1}{4}x^4 - \frac{4}{3}x^3 + 3x + c$ , karena $f(0) = 1$, maka
→$f(0) = \frac{1}{4}(0)^4 - \frac{4}{3}(0)^3 + 3(0) + c = 1$
→$f(0) = 0 - 0 + 0 + c = 1$
→    $= c = 1$, berarti c = 1
Jadi, nilai $f(x)$ adalah $f(x)$ =$\frac{1}{4}x^4 - \frac{4}{3}x^3 + 3x + 1$
